Introducing The Cloud IP WAN Simulator

When designing your Intranet or an Internet Site the user's experience cannot be determined accurately. Despite the investment of time, money and effort, the end result may be be a frustrated user. Evaluating response times from a variety of users, with a wide range of Wide Area Network (WAN) connections, is an almost impossible task.

Enter The Cloud

The Cloud is a software tool that enables you to experience your Intranet applications or  Internet site just as a prospective user would under all possible network conditions.

The Cloud does this using a unique Network Emulation methodology. It uses your local network infrastructure introducing WAN network parameters between The Cloud host machine and it's NIC (Network Interface Card). The user has complete control over these WAN parameters. They can be set manually to predetermined performance parameters or  by Playback of performance recordings of any real WAN connection.

Features:

  • Software product - uses the installed Network Interface Card (NIC) for LAN interface
  • Simple yet powerful setup, giving the user full control over the simulator
  • Simulate any combination of Bandwidth, Latency and Error rate
  • Includes Network Analyzer giving real time statistics
  • Low host CPU usage
  • Standalone, real time, real connection performance recording module included
  • Real time, real connection performance Playback through the simulator
  • Random Playback through the simulator (creates worse case conditions)
  • Works on NT 4.0 Workstation or Server
