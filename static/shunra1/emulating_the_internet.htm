The Internet Network

Internet network parameters are constantly changing. The way the Internet behaves doesn't conform to any known statistical rule. To illustrate these fluctuations one need only monitor the bandwidth available to a single connection and how it changes throughout the day.

Simulating The Internet

The ideal way to simulate the Internet is to record the actual Internet performance and use this recording as a basis for the simulation thereby eliminating statistical extrapolations. The Cloud simulates the Internet using this Record & Playback mechanism.

Network Recording Files

Playback dynamics are based on actual network Recording files. To create these recordings the user specifies the destination IP address or URL (Uniform Resource Locator) and the time scale.
