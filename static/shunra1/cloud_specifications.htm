The Cloud is a Software product operating on Windows NT Systems. The Cloud host can be connected to a Local Area Network in one of the following configurations:

Installed on a server    
The Cloud simulates the connection parameters of a server over a WAN. 

Installed on a number of clients The Cloud gives the user the ability to simulate different performance parameters for each client connected to the server through each individual simulatedWAN connection.

Installing The Cloud with multiple Network Interface Cards (NICs).  
The result is a simulated WAN in which two or more LAN segments are connected to it.

 

The Cloud Minimum Host System Requirements:
 

Microsoft Windows NT 4.0 Service Pack 3.
Pentium 133MHz or higher.
32 Mbytes RAM or more.
5 Mbytes free disk space.
